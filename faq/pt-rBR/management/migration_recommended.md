# MIGRAÇÃO RECOMENDADA

Parece que esta carteira foi criada usando uma lista de palavras que não são do inglês e/ou contém símbolos na frase secreta diferentes dos seguintes:

A-Z a-z 0-9 '"`&/?!:;.,~*$=+-{}<>\_#@|%

Se o saldo e as transações da sua carteira forem exibidos corretamente, nenhuma ação será necessária. No entanto, se você não vir saldo ou transações, siga estas etapas para restaurar sua carteira usando a opção de restauração não padrão:

1. Vá para a guia Configurações.
2. Selecione Gerenciar carteiras.
3. Clique no botão Restaurar.
4. Escolha Restauração não padrão.
5. Siga as instruções de restauração.

Assim que a restauração for concluída, seu saldo e transações deverão estar visíveis.

Após esse processo, o aplicativo de carteira exibirá outro alerta com as etapas recomendadas para transferir seus fundos para uma carteira compatível com o padrão.
